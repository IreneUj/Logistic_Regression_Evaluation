# Logistic_Regression_Evaluation
A machine learning project analyzing airline passenger satisfaction using Python, exploratory data analysis (EDA), feature engineering, and logistic regression.
# Project Overview
This project analyzes airline passenger satisfaction using machine learning techniques. The objective is to identify the factors that most influence passenger satisfaction and build a Logistic Regression model capable of predicting whether a passenger is satisfied or dissatisfied based on flight experience and service quality.
The project includes data cleaning, exploratory data analysis (EDA), feature engineering, model building, model evaluation, and interpretation of results.

# Business Problem
Airlines aim to improve customer experience and increase passenger loyalty. Understanding which aspects of the travel experience have the greatest impact on passenger satisfaction enables airlines to prioritize service improvements and make data-driven business decisions.

# Objectives
Perform exploratory data analysis on airline passenger data.
Clean and preprocess the dataset.,
Handle missing values and encode categorical variables.,
Build a Logistic Regression model to predict passenger satisfaction.,
Evaluate model performance using classification metrics.,
Identify the key factors influencing passenger satisfaction.,
Provide actionable recommendations based on the findings.

# Dataset
The dataset contains information about airline passengers, including demographic characteristics, travel details, service ratings, delays, and satisfaction status.
